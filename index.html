Project idea

Overview

My project idea would be a mobile application which centralises all chats from the messaging portion of social media apps this includes Facebook, Instagram, Wechat, Snapchat, Whatsapp etc, with this application it aims to create a one stop place to read and send messages, where an individual can just access all their messages through this singular app to alleviate the need to open multiple applications, it will be a simplified area to access messages in one area to decrease the clutter of having multiple apps.

Motivation

My motivation for this project would be due to me wanting to message with only one application rather than needing to have multiple applications open at once, like with moving all types of cards on google pay, I believe that having chats centralised will diminish clutter with the amount of chats and of social medias platforms a user may be using, the fact is that having multiple applications is becoming fatiguing and running multiple at once can be a drain on the battery life and storage of a mobile device, and by having one application do all the messaging it would minimise the phone utilisation and clutter of applications from the phones front menu, solving my issues with battery drain and messy user interfaces caused by large amount of application installs on a device.

Description

• Clean up your cluttered icon spaces and reduce battery usage by using this app.
• This application will allow users to express themselves through their desired social media within a centralised realm, this means that messaging will all be located within this application and allows the user to converse with people faster without the need to open and close multiple applications to simply chat.
• This application will allow users to express themselves through their desired social media within a centralised realm, this means that messaging will all be located within this application and allows the user to converse with people faster without the need to open and close multiple applications to simply chat.
• When utilising this application users will be prompted to login to each social media that they want to unify the chats of, this can be as many as they want and even just one if they have chosen so, however the chats would be opened through a browser-based way.
• When browsing the app each chat should be colour coded and icon highlighted to indicate a chat from a different social media.
• All notifications will be alerted through this application rather than the corresponding social media app and when opened it will navigate to the correct chat that was selected from the notification.
• It will have a hamburger drop down menu by the side of the screen to allow users to select the social media providers and once chosen will show each chat that is associated with that social media.
• There is also a sort function for a-z/z-a and ascending/descending order for each social media title making it easier for the user to find which social media chat they would like to talk in.
• The colours and layout will be minimal and simplistic for the user to understand where each button navigates as well, giving them less visual distraction for where to go.
• Each social media button will have their original application icon and their primary colour to highlight which social media however this can be changed by the user if preferred ultimately to help the user navigate without error and get to their desired chat efficiently.
• When reopening the application, it should continue where the user had left off keeping the previous chat open until the user has chosen to back out into the menu for social media selection.
• The application also allows users to also customise the font size of the texts for better viewability and whether to get notifications from each social media, this will give users greater control of what notifications they want to see from each social media.
• Navigating the application will happen in a horizontal and vertical fashion showing lists of social medias in a rectangular bubble button, each button will have a drop-down menu to customise each social media colour and can be reorganised by holding down on the icon and moving it.
• When a chat is opened it will look identical to respective social medias chat layout and cannot be modified unless the social media provider has allowed their chats to do so.

Tools and Technologies

Tools required to create the application would be android studio with java installed on a computer, this is used to code and mock-up designs for how the application will look before running on the phone, android studio is needed as it allows for the program to be made for specifically android devices and not any other mobile based OS, hardware needed would be an android device and a computer that is able to run the android studio and has an output USB port so it can be uploaded to an actual android device, the computer may also need to be running a 64bit operating system to be able to compile the app.

Skills required

Skills required to create the program would be programming background knowledge mainly in C language under JavaScript, to gain the skills needed to program the application multiple online courses should suffice in creating a simple user interface and most basic functions like login and chat selection, furthermore some YouTube videos covering the basics of programming an app may be enough to develop a working concept of the application. Feasibility of accessing hardware needed is extremely high as computers and android devices have become common and easy to acquire.

Outcome

if this project was successful, I believe that people would start to minimise multiple apps being installed just for messaging and make them only keep ones for more advanced functions like image posting, not needing to have multiple apps for messaging would also simplify user’s choice of social media and allow them to chat to more people in one area without having to search and find the other application that would normally be used overall reducing clutter on their device and making it more efficient to message, hopefully making more people utilise a single application over multiple. The problem of having multiple apps for messaging will be solved if people commit to using one app to load all their messages and make more individuals want it over wasting precious storage and battery on their mobile device.
